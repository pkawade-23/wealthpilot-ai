# WealthPilot AI

WealthPilot AI is an AI-powered personal finance platform that automatically builds a financial profile from emails, documents, and user data, providing personalized insights, intelligent financial planning, and natural language interaction.

## Planned Features

### Financial Inbox
- Email forwarding
- PDF uploads
- Receipt uploads
- AI-powered document understanding

### Personal Finance
- Expense tracking
- Income tracking
- Investment portfolio
- Loan management
- Net worth dashboard

### AI Features
- AI Financial Assistant
- Financial Q&A
- Spending Insights
- Goal Planning
- Financial Scenario Simulator

### Engineering
- REST APIs
- JWT Authentication
- Docker
- CI/CD
- Clean Architecture

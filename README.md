# WealthPilot AI

WealthPilot AI is an AI-powered personal finance platform that automatically understands financial emails and documents to provide personalized financial insights.

## Features

- AI Financial Inbox
- Document Intelligence
- Expense Tracking
- Investment Dashboard
- AI Financial Advisor
- Financial Scenario Simulator

## Tech Stack

### Frontend
- Angular

### Backend
- FastAPI

### Database
- MongoDB

### AI
- Ollama
- LangChain
- ChromaDB

### DevOps

- GitHub Actions
